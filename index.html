What's Happening?

Every push to main triggers GitHub Actions which SSHes into your EC2 instance, copies files, and restarts Nginx automatically.

EC2 One-time Setup

Run on your instance:

sudo apt update && sudo apt install nginx -y

Then open port 80 in your Security Group.
